There are 10 html files and the readme file in the zip folder.

To start with the execution of the assignment, click on any of the html file which opens up the graph of that file.

To navigate to the visualization of other players, just click on links that are displayed on the page that is being currently displayed and you will be navigated to the respective player's visualization webpage.

The starting point of the data visualization is the below link:
http://www.public.asu.edu/~hathakka/DataVisualization/Roger%20Federer.html
