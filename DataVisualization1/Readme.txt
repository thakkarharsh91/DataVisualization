README

The entry page is index.html which calls two files histogram.html and piechart.html.
The link to tne entry page is : http://www.public.asu.edu/~hathakka/index.html

Histogram 1 features :
Drag the bar from left to right to see the data of all the users

Histogram 2 features :
Hover the mouse over the bar strip to get the information in details

Pie chart features :
Hover the mouse over the pie section to get the count of the particular intention and the percentage distribution
Click on the small square located inside the cirlce to show/hide the particular intention from the pie chart.
